# WEAVE Public Packet

Public appendix packet for the Livepeer WEAVE proposal.

WEAVE is an open-source, semi-autonomous agentic orchestration tool designed to turn intent into researched, engineered, tested, maintained, and distributed workloads on Livepeer. It introduces three perpetual economic incentive packets (Workload Facilitator Hackathon, App Developer Incentive, and Orchestrator Incentive) to align all network participants.

This repository contains the public review documents only:

- `features/livepeer-proposal-v6/GLOSSARY.md`
